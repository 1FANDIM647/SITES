This page written with Java Script implements a Clock. The interface is a single function that takes no arguments:

Clock() Also available is the Server Clock. Where you can set the time to whatever area that you like. (eg paris and new york) See my example page of Times Around the World.

Also available is the Status Clock which puts the time in the menu bar.

Warning

I have found Java Script to be quite unstable, when playing with it, Netscape 2.0 crashed many times. And consitently too..

How To

  • Include clock.txt in the HEAD section of your document.
  • Add the following inside your document: 
    <script language="javascript">
Clock();
</script>
  • View the source of this document for an example.
Of course you really should comment out the java script to other browsers.
